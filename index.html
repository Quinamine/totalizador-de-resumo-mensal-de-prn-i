As células dos totais têm o acesso interdito para assegurar que os seus resultados não sejam acidentalmente modificados.

Por que algumas células ficaram com o fundo vermelho?

Toda célula cujo fundo está vermelho tem um ou mais algarismos omissos por indisponibilidade de espaço na respectiva célula.

As células de entrada de dados do Totalizador têm uma largura para 9 algarismos no máximo, portanto, só podem mostrar valores até 999 999 999. A partir de 1 000 000 000, ficam visíveis somente os primeiros 9 algarismos e do 10º em diante, ficam fora da área visível da célula.

Presumo que actualmente (2023) não haja uma Unidade Sanitária em Moçambique com um número de consultas de PRN I maior que 999 999 999.

Ao utilizar o Totalizador de Resumo Mensal de PRN I, o usuário concorda com o armazenamento local de dados, a fim de melhorar a sua experiência de utilização.

Sobre o totalizador

O Totalizador de Resumo Mensal de PRN I é um serviço online gratuito que auxilia na elaboração do respectivo resumo totalizando-o automaticamente com base nos dados preenchidos pelo usuário (Profissional de Saúde). A sua estrutura é baseada na da ficha de resumo mensal de PRN I actualmente (2023) vigente no Serviço Nacional de Saúde em Moçambique.

O propósito é de flexibilizar a elaboração do resumo através do cálculo automático dos totais.

O Totalizador de Resumo Mensal de PRN I reúne os seguintes recursos:

  • Calcula e autopreenche as células dos totais;
  • Fornece resultados verdadeiros;
  • Flexibiliza a elaboração do resumo;
  • Salva os dados no navegador do usuário até este excluí-los;
  • Pode ser impresso.

Política de Armazenamento de dados

Web Storage é um recurso para armazenamento local (no navegador do usuário) de informações por meio de pequenos arquivos de texto, útil para guardar predefinições.

Existem 2 tipos de Web Storage:

  • Session Storage (Armazenamento de Sessão) – elimina os dados gravados quando a janela da página é fechada.
  • Local Storage (Armazenamento Local) – mantém os dados salvos por um período determinado ou até que o usuário exclua.

O Totalizador de Resumo Mensal de PRN I utiliza o primeiro para guardar o consentimento do usuário à Política de Armazenamento e o último pelas razões do subtítulo a seguir.

Por que armazeno dados?

Por 2 razões, enumeradamente:

Fazer um backup local
Fazer uma cópia de segurança dos dados da ficha no navegador do usuário. Se forem preenchidos, os dados que são salvos incluem:
  • Os dados da ficha propriamente dita.
  • Dados adicionais como:
    • Nome da Unidade Sanitária
    • Código da Unidade Sanitária
    • Nome do Sector da Unidade Sanitária
    • Data do Relatório
    • Nome do Responsável pelo compilamento;
    • Província;
    • Distrito;
    • Mês/Ano;
Durabilidade dos dados
Manter salvos os dados previamente inseridos no acto de preenchimento da ficha por tempo suficiente para serem aproveitáveis. Isso é importante, pois possibilita ao Profissional interromper sem o risco de voltar à estaca zero, a elaboração do resumo para continuar num outro momento oportuno, sobretudo se ainda lhe faltar dados de alguns indicadores para preencher nas respectivas células da ficha.

Destino dos dados

Todos os dados são salvos localmente (no navegador do usuário). O armazenamento local é para fins de memorização e com isso tornar os dados suficientemente duráveis.

Gerenciamento dos dados pelo usuário

No menu do Totalizador, existe a opção “Esvaziar a ficha”, que permite ao usuário apagar todos os dados da ficha salvos pelo Totalizador na Web Storage.

A aceitação dos termos de armazenamento apresentados neste bloco é dada pela utilização do Totalizador de Resumo Mensal de PRN I.

Como salvar a ficha em PDF?

  1. Clique na opção "Imprimir".
  2. Na janela que abrirá, no campo de Impressora ou Destino selecione a opção “Salvar como PDF”.
  3. No campo de Layout ou Orientação, selecione "Paisagem". Isso é para que toda ficha caiba no papel A4 e não haja omissão das últimas colunas.
  4. Por fim, clique em salvar e escolha o destino no seu computador.

Se no navegador Firefox haver omissão das últimas colunas no modo paisagem, teste no modo retrato.
